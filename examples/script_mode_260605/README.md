# Exemple d'utilisation en mode script

Cet exemple présente l'utilisation principale du modèle CGN depuis une
configuration YAML.

- `config.yaml` : scénario de référence ;
- `run.py` : workflow utilisateur recommandé avec `Vessel.run()` ;
- `run_detailed.py` : workflow décomposé pour l'inspection et le débogage ;
- `results.csv` : exemple de résultats filtrés.

La présentation détaillée de cet exemple est disponible dans
[`docs/example_script.md`](../../docs/example_script.md).

Pour la référence complète de la configuration YAML, consulter
[`docs/script_guide.md`](../../docs/script_guide.md).
